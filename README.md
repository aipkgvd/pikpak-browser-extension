## 常見問題（FAQ）

**Q：PikPak 瀏覽器擴充支援 Edge 嗎？**

完全支援。Edge 使用 Chromium 核心，安裝方式和 Chrome 完全一樣，在 Edge 的附加元件商店搜尋「PikPak Browser Extension」即可。

**Q：擴充裝了但頁面上沒出現 PikPak 圖示？**

可能是頁面的連結格式特殊或做了防抓取處理，也可能是頁面需要重新整理才能觸發辨識。試著 F5 重整一次頁面，或者改用右鍵選單方式操作。此外，如果網頁用了 JavaScript 動態產生連結，擴充有時會需要等待一下才能識別。

**Q：免費版的雲端下載限制是怎麼算的？**

免費版的下載次數和可用空間都有上限，具體數字依 PikPak 當前政策為準（可在 App 或網頁版確認）。用完配額需要等待重置，或升級 Premium 取得無限次數。

**Q：我的 IP 位址和下載記錄會被 ISP 看到嗎？**

不會。整個雲端下載流程發生在 PikPak 的伺服器上，BT 網路和 ISP 只能看到 PikPak 的伺服器 IP，無法追蹤到你的個人資訊。PikPak 所有伺服器通訊均使用加密協議。

**Q：Premium 到期後資料會怎樣？**

到期後帳號回到免費版狀態，可繼續存取現有檔案。若儲存量超過免費版配額（6 GB），帳號進入「超額使用狀態」，暫時無法新增內容，但已有檔案在一段寬限期內仍安全，重新訂閱即可恢復正常。
